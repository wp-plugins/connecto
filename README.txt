== Description ==

Connecto lets you add targeted lead forms and notifications on your websites.
With Connecto's bars and notifications you can increase your site's conversion
rate significantly. Connecto is free in beta.

Main features include:
* Fully customizable top/bottom bar
* Trigger when a user has scrolled to the bottom of the page
* Trigger based on time spent on page/session
* Trigger based on Geolocation
* Many positions available for positioning
* Lead/Email subscription form in the bar
* Analytics Dashboard
* Email on submission
* Submissions Dashboard with details like location, referal, source etc

Take an online demo to get an idea of how the feedback tab, survey windows and
notification messages will look on your website -
[www.connecto.io/](http://www.connecto.io/)

PS: You'll need register for a [Connecto API key](http://www.connecto.io/accounts/register)
to use it.

== Installation ==

After activating the plugin there shall be a "Connecto" admin page on your admin
left bar.


= Installing The Connecto Plugin =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Connecto'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `connecto.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `connecto.zip`
2. Extract the `connecto` directory to your computer
3. Upload the `connecto` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

== Frequently Asked Questions ==

= How do I get started ? =
1. Register with Connecto.
2. From your Connecto Profile page, generate API key and paste it in your wordpress admin
3. From your wordpress admin or from the Connecto admin, you can launch notification bars, lead form bars etc.


= Is it free? =

Connecto is free in Beta. We also have a free forever plan for upto a certain
number of pageviews.
